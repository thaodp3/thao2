# BuzzWords 🍃 Word Scramble Game

A juicy, fruit-themed word scramble game for Digital Marketing & E-commerce pros.

## Topics Covered
- Digital Marketing
- E-commerce
- TikTok
- Shopee
- Meta Ads

## How to Play
1. Open `index.html` in your browser (no server needed)
2. Unscramble the word shown
3. Type your answer and press Submit or Enter
4. Use Hint for a clue (costs 5pts instead of 10pts)
5. Build a streak for bonus bragging rights!

## Tech Stack
- Vanilla HTML, CSS, JavaScript
- Google Fonts: Fredoka One + Nunito
- localStorage for leaderboard persistence
- Pure CSS animations
- Hand-crafted SVG illustrations

## Run Locally
Just open `index.html` — no build step required.

## Customization
Add words in `js/game.js` in the `wordBank` array.
Each word needs: `word`, `hint`, `category`.
